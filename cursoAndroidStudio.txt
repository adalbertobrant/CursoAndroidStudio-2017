LAYOUTS

Arquivos XML na pasta /res/layout

FrameLayout
- Componente para ocupar a tela inteira
-layout mais básico

ATRIBUTOS

-layout_width - largura
-layout_height - altura "wrap_content" --> apenas o necessário , match_parent --> todo o conteúdo
-background - cor de fundo
-layout_gravity - centro de gravidade

id="@+id/imageView" --> 

LinearLayout
-segue uma lógica linear horizontal ou vertical, seguindo um atributo "orientation"

ATRIBUTOS

-orientation
-layout_height
-layout_width
-layout_gravity
-layout_weight - coloca um peso (10% ou mais e ajusta a tela)

RelativeLayout
- permite posicionar m componente relativo a outro (abaixo, acima ou ao lado )

ATRIBUTOS

CoordinatorLayout

É um container - aceita vários componentes - pode coordenar vários componentes
Faz a ancoragem de botões na tela e sobe junto com a tela .

Activity

Todos os atributos vistos foram em XML 
Cada tela se chama activity
O XML é estático e mostra ao usuário visualização
O java controla o XML
